## 🐾 Rakkly Protectoras

### Tecnología para que las protectoras dediquen menos tiempo a gestionar y más tiempo a cuidar animales

**Rakkly Protectoras** es una plataforma integral para asociaciones y protectoras de animales.

Permite centralizar en un único lugar:

* Fichas de animales y seguimiento de su estado
* Web pública de animales en adopción
* Solicitudes de adopción organizadas por fases
* Inventario de alimentos, medicación y materiales
* Avisos públicos de productos necesarios
* Gestión de tareas y responsables
* Voluntarios, socios, padrinos y donantes
* Informes para memorias, financiación y subvenciones

Cada protectora dispone de su propio espacio privado y de una página pública desde la que las personas pueden adoptar, colaborar, hacerse socias, apadrinar animales o participar como voluntarias.

[Visitar Rakkly Protectoras](https://protectoras.rakkly.com)
